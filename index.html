Jquery-canvas-charts

This is a jQuery plugin that creates Charts on Canvas an HTML5 object

Your browser does not support the canvas element.

JavaScript

  1. Include Jquery
  2. Include the plugin
  3. Use the plugin

Following is a sample

$('#browserTable').canvasCharts('barChart',{ canvasId:'Canvas1', barWidth:20, barGap:30, steps:7 }); $('#browserTable').canvasCharts('pieChart',{ canvasId:'Canvas2' }); $('#browserTable').canvasCharts('lineChart',{ canvasId:'Canvas3', steps:7 });
